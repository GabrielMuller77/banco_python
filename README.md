# Sistema Bancário simples em Python.

* main -> Sistema principal responsável por rodar o programa.
* sistema -> Cérebro do programa, responsável por todas as operações.
* menu -> Parte visual, responsável pela parte visual e ler a escolha do usuário.
* arquivo -> Responsável por ler e criar arquivos JSON.
* validacoes -> Responsável por validar o nome do usuário, apenas para letras.
* conta -> Contém classes abstratas e define métodos de pagamentos.
* conta_corrente -> Conta que herda funções abstratas da classe mãe e possui suas próprias especificações.
* conta_poupanca -> Conta que herda funções abstratas da classe mãe e possui suas próprias especificações.
